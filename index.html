Sobre Este Projeto

Este projeto demonstra o uso de HTML semântico e CSS avançado, sem JavaScript ou outras tecnologias externas. O objetivo é mostrar domínio técnico completo através de:

HTML Puro

  1. Semântica rigorosa: Uso correto de elementos estruturais
  2. Acessibilidade: Suporte a tecnologias assistivas
  3. Padronização: Conformidade com W3C
  4. Completude: Demonstração de todos os elementos relevantes

CSS Moderno

  1. CSS Grid & Flexbox: Layouts modernos e responsivos
  2. Custom Properties: Sistema de design tokens
  3. Animações CSS: Transições e keyframes avançadas
  4. Design System: Paleta de cores e tipografia consistente

Características Técnicas

Acessibilidade

  • Textos alternativos em todas as imagens
  • Rótulos associados a todos os campos de formulário
  • Teclas de acesso (accesskey) para navegação
  • Estrutura hierárquica de cabeçalhos
  • Suporte a leitores de tela
  • Contraste adequado de cores

Semântica HTML

  • Elementos estruturais adequados (header, nav, main, section, footer)
  • Marcação semântica para conteúdo (article, aside, figure)
  • Tabelas com escopo e legendas apropriadas
  • Formulários com agrupamento lógico

CSS Moderno

  • Sistema de design com CSS Custom Properties
  • Layout responsivo com CSS Grid e Flexbox
  • Animações e transições suaves
  • Tipografia escalável e legível
  • Paleta de cores profissional
  • Performance otimizada

Responsividade

  • Mobile-first design
  • Breakpoints bem definidos
  • Imagens e mídias adaptáveis
  • Navegação colapsível
  • Tipografia fluida
  • Touch-friendly interfaces
