Introdução ao CSS Moderno

CSS (Cascading Style Sheets) é a linguagem responsável pela apresentação visual de documentos HTML. Este guia demonstra conceitos avançados e modernas práticas de desenvolvimento CSS.

Características deste Projeto CSS

Funcionalidades Implementadas

  • Custom Properties (CSS Variables) - Sistema de design tokens
  • CSS Grid - Layout moderno e responsivo
  • Flexbox - Alinhamento e distribuição
  • Animações CSS - Transições e keyframes
  • Media Queries - Design responsivo
  • Pseudo-elementos - Efeitos decorativos

Metodologias Aplicadas

  • Mobile-First - Desenvolvimento responsivo
  • BEM-like - Nomenclatura consistente
  • Design Tokens - Variáveis CSS organizadas
  • Progressive Enhancement - Funcionalidades graduais
  • Accessibility - Suporte a tecnologias assistivas
  • Performance - Otimização e eficiência

1. Seletores CSS Avançados

Tipos de Seletores

Demonstração de Seletores CSS
Seletor Sintaxe Descrição Exemplo
Universal * Seleciona todos os elementos * { margin: 0; }
Elemento elemento Seleciona elementos por tag h1 { color: blue; }
Classe .classe Seleciona elementos por classe .card { padding: 1rem; }
ID #id Seleciona elemento por ID único #header { position: fixed; }
Atributo [attr] Seleciona por atributo [required] { border: red; }
Pseudo-classe :pseudo Estados especiais a:hover { color: red; }
Pseudo-elemento ::pseudo Partes específicas p::first-line { font-weight: bold; }

Exemplo Prático - Seletores Combinados

/* Seletor descendente */
.card p {
    margin-bottom: 1rem;
}

/* Seletor filho direto */
.nav-list > li {
    display: inline-block;
}

/* Seletor irmão adjacente */
h2 + p {
    font-size: 1.125rem;
}

/* Seletor de atributo com valor */
input[type="email"] {
    background-image: url(email-icon.svg);
}

/* Pseudo-classes estruturais */
tr:nth-child(even) {
    background-color: var(--gray-50);
}

/* Pseudo-elementos avançados */
.btn::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: linear-gradient(45deg, transparent, rgba(255,255,255,0.1), transparent);
    transform: translateX(-100%);
    transition: transform 0.6s;
}

.btn:hover::before {
    transform: translateX(100%);
}

2. Box Model e Dimensionamento

O Box Model é fundamental para entender como os elementos são renderizados no CSS.

Componentes do Box Model

CONTENT
Conteúdo do elemento
PADDING - Espaçamento interno
BORDER - Borda do elemento MARGIN - Espaçamento externo

Propriedades de Dimensionamento

/* Box Sizing */
* {
    box-sizing: border-box; /* Inclui padding e border no width */
}

/* Dimensões básicas */
.element {
    width: 100%;
    max-width: 1200px;
    min-width: 320px;
    height: auto;
    min-height: 100vh;
}

/* Margens e Padding */
.spacing-example {
    margin: 2rem 1rem 1.5rem 1rem; /* top right bottom left */
    padding: 1rem; /* Todos os lados iguais */
    margin-inline: auto; /* Centralização horizontal moderna */
    padding-block: 2rem; /* Padding vertical lógico */
}

/* Overflow */
.container {
    overflow: hidden; /* Esconde conteúdo excedente */
    overflow-x: auto; /* Scroll horizontal quando necessário */
    overflow-y: scroll; /* Scroll vertical sempre visível */
}

3. Sistemas de Layout Modernos

CSS Grid - Layout Bidimensional

Grid Item 1
Grid Item 2
Grid Item 3
/* CSS Grid Básico */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 colunas iguais */
    grid-template-rows: auto;
    gap: 1rem;
}

/* Grid Avançado com Áreas */
.layout-grid {
    display: grid;
    grid-template-areas: 
        "header header header"
        "sidebar main aside"
        "footer footer footer";
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

/* Grid Responsivo */
.responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 2rem;
}

Flexbox - Layout Unidimensional

Flex Item 1
Flex Item 2 (2x)
Flex Item 3
/* Flexbox Básico */
.flex-container {
    display: flex;
    justify-content: space-between; /* Distribuição horizontal */
    align-items: center; /* Alinhamento vertical */
    gap: 1rem;
}

/* Flex Items */
.flex-item {
    flex: 1; /* Crescimento igual */
    flex-basis: 200px; /* Base mínima */
    flex-shrink: 0; /* Não encolhe */
}

/* Direções e Wrap */
.flex-column {
    flex-direction: column;
    flex-wrap: wrap;
    align-content: flex-start;
}

/* Alinhamento Individual */
.flex-item-center {
    align-self: center;
}

/* Flexbox para Componentes */
.card-flex {
    display: flex;
    flex-direction: column;
    height: 100%;
}

.card-content {
    flex: 1; /* Ocupa espaço disponível */
}

.card-footer {
    margin-top: auto; /* Empurra para baixo */
}

/* Sistema de Tipografia com Variáveis */
:root {
    --font-primary: 'Inter', system-ui, sans-serif;
    --font-secondary: 'JetBrains Mono', monospace;
    --font-display: 'Poppins', var(--font-primary);
    
    /* Escala tipográfica modular (1.25 - Major Third) */
    --text-xs: 0.75rem;
    --text-sm: 0.875rem;
    --text-base: 1rem;
    --text-lg: 1.125rem;
    --text-xl: 1.25rem;
    --text-2xl: 1.5rem;
    --text-3xl: 1.875rem;
    --text-4xl: 2.25rem;
    --text-5xl: 3rem;
}

/* Tipografia responsiva */
h1 {
    font-size: clamp(2rem, 4vw, var(--text-5xl));
    line-height: 1.1;
    letter-spacing: -0.02em;
}

/* Propriedades avançadas */
.text-optimized {
    font-variant-numeric: oldstyle-nums;
    font-feature-settings: "liga" 1, "kern" 1;
    text-rendering: optimizeLegibility;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
}

/* Texto responsivo */
.responsive-text {
    font-size: clamp(1rem, 2.5vw, 1.5rem);
}

/* Limitar largura para legibilidade */
.readable-text {
    max-width: 65ch; /* 65 caracteres */
    line-height: 1.6;
}

/* Keyframes para animações */
@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

@keyframes slideInRight {
    from {
        opacity: 0;
        transform: translateX(50px);
    }
    to {
        opacity: 1;
        transform: translateX(0);
    }
}

@keyframes pulse {
    0%, 100% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.05);
        opacity: 0.8;
    }
}

/* Aplicação de animações */
.animate-fade-up {
    animation: fadeInUp 0.6s ease-out;
}

.animate-slide-right {
    animation: slideInRight 0.8s ease-out;
}

.animate-pulse {
    animation: pulse 2s ease-in-out infinite;

/* Animações com delay */
.stagger-animation:nth-child(1) { animation-delay: 0.1s; }
.stagger-animation:nth-child(2) { animation-delay: 0.2s; }

7. Design Responsivo

Breakpoints e Media Queries

Sistema de Breakpoints
Dispositivo Largura Mínima Media Query Características
Mobile 320px - 767px @media (max-width: 767px) Layout vertical, navegação colapsada
Tablet 768px - 1023px @media (min-width: 768px) Layout híbrido, 2 colunas
Desktop 1024px+ @media (min-width: 1024px) Layout completo, múltiplas colunas 
/* Mobile First Approach */
.responsive-grid {
    display: grid;
    grid-template-columns: 1fr; /* Mobile: 1 coluna */
    gap: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .responsive-grid {
        grid-template-columns: repeat(2, 1fr); /* 2 colunas */
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .responsive-grid {
        grid-template-columns: repeat(3, 1fr); /* 3 colunas */
    }
}

/* Queries avançadas */
@media (orientation: landscape) {
    .landscape-layout {
        flex-direction: row;
    }
}

@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
    }
}

/* Container queries (futuro) */
@container (min-width: 400px) {
    .card {
        display: flex;
        flex-direction: row;
    }
}

/* Unidades responsivas */
.fluid-typography {
    font-size: clamp(1rem, 2.5vw, 2rem);
    margin: clamp(1rem, 5vw, 3rem) 0;
}

/* Imagens responsivas */
.responsive-image {
    width: 100%;
    height: auto;
    object-fit: cover;
    aspect-ratio: 16 / 9;
}
